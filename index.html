Nostalgia: Desenhos da Infância

Este projeto celebra os desenhos animados que marcaram minha infância, com uma galeria interativa dos personagens mais memoráveis e um quiz nostálgico para testar seus conhecimentos!

No quiz, você responde perguntas como:

  • Qual é o nome do melhor amigo do Bob Esponja?
  • De qual planeta vem o Goku?
  • Qual é o alimento favorito do Garfield?

No final, você recebe uma pontuação e sugestões de novos desenhos clássicos para assistir!
